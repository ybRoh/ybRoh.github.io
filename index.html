제품 소개

자동차부품 제조 현장에 필요한 4가지 핵심 시스템을 제공합니다.

MES 생산관리

사출/프레스/CNC/연마 공정에 최적화된 생산관리 시스템. 작업지시, 생산실적, LOT 추적, OEE 분석, 설비보전까지 통합 관리합니다.

  • 작업지시 및 생산실적 관리
  • LOT 추적 (정방향/역추적)
  • OEE 분석 및 실시간 KPI
  • 설비보전 (PM) 관리
  • 금형 수명 관리
Python Streamlit PostgreSQL
다운로드 GitHub

IATF 16949 품질관리 (QMS)

자동차 산업 품질경영시스템 IATF 16949 규격에 맞춘 품질관리 시스템. SPC, FMEA, MSA, PPAP, APQP 등 핵심 도구를 지원합니다.

  • SPC 공정능력 분석 (Cp/Cpk)
  • FMEA 위험분석
  • MSA 측정시스템 분석 (GR&R)
  • PPAP / APQP 관리
  • 감사 및 교육훈련 관리
FastAPI Vue 3 Element Plus PostgreSQL
다운로드 GitHub

MES 웹 시스템

React + Node.js 기반의 풀스택 MES 웹 애플리케이션. 기준정보, 생산계획, 재고관리, 품질검사, 실시간 대시보드를 제공합니다.

  • 기준정보 관리 (품목/BOM/공정/설비)
  • 생산계획 및 작업지시
  • 재고 입출고 관리
  • 품질검사 등록
  • 실시간 대시보드
React 18 Node.js Prisma PostgreSQL
다운로드 GitHub

소모품 통합구매

공장 소모품의 구매요청부터 발주, 입고, 재고관리까지 전 과정을 통합 관리하는 구매관리 시스템입니다.

  • 구매요청 및 승인 워크플로
  • 발주 및 납기 관리
  • 입고 검수 처리
  • 소모품 재고 현황
  • 구매 이력 분석
Python Flask SQLite
다운로드 GitHub

핵심 기능

자동차부품 제조 현장의 요구사항을 반영한 핵심 기능을 제공합니다.

실시간 KPI

생산수량, 불량률, OEE, 가동률 등 핵심 지표를 실시간 대시보드로 모니터링합니다.

IATF 16949

SPC, FMEA, MSA, PPAP, APQP 등 자동차 산업 품질 Core Tool을 통합 지원합니다.

LOT 추적

원자재부터 완제품까지 양방향 LOT 추적으로 품질 이슈 발생 시 신속하게 대응합니다.

설비보전

PM 계획, 보전 이력, 비가동 관리로 설비 가동률을 극대화하고 돌발 고장을 예방합니다.

자재관리

자재 입고, FIFO 재고관리, BOM 기반 자동 차감으로 정확한 재고를 유지합니다.

권한관리

역할 기반 접근 제어(RBAC)로 사용자별 메뉴 접근 권한을 세밀하게 관리합니다.
